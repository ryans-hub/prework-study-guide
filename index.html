HTML

  • The head element contains information about the webpage. This element contains metadata, which is not visible to the enduser.
  • The body element represents the visible content shown to the user. The elements placed inside are what the user sees and interacts with.

CSS

  • CSS (Cascading Style Sheets) is how web content is presented to users - defines the colors, aesthetics, visual layout, and organization.
  • There are 3 ways of styling a web page: Inline CSS, Internal CSS Style sheet, and External CSS Style sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Inline CSS

  • Inline CSS uses a style element directly in the HTML file, and only applies to a specific HTML tag. This is quick and permanent, but less flexible than the other methods of styling a web page.

Internal CSS Style sheet

  • Embeds an entire stylesheet directly in the HTML file using a style element in the head element.
  • The CSS rules apply to all tagged page elements

External CSS Style sheet

  • Most common way to style a webpage because it organizes all CSS rules in a seperate CSS file. This makes design changes easier.
  • Requires linking the CSS file using a link element placed in the HTML's head element
  • CSS rules apply to all tagged page elements, similar to Internal CSS Style sheet CSS rules

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: add command addsc changes in the current feature branch to the staging area. The -A flag means we add all changes. The staging area informs Git the changes will be added to the next commit.
  • git commit -m "x": contains commit message describing what the commit contains.
  • git pull origin main: pulls branch changes from main branch into our local environment, syncing local branch with the remote main branch.
  • git push origin x/x: pushes local changes to remote main branch. Can create new branch or add changes to existing branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. JS executes code in a top down fashion.
